# POD Catch, Apps Script

Paste Code.gs into each Google Sheets' Apps Scripts. 

Make sure to find & replace LPODD0 in the script. There are 3 instances:

  (1) Line 4 - replace with relevent _Event ID_ 

  (2) Line 10 - replace with relevent _Event ID_ 

  (3) Line 82 - replace with your **PODID** (i.e. LPODD1, ZPODO1) 

Find & Replace YOUR-EMAIL to then replace the address
